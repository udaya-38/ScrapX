# ScrapX — Enterprise Scrap Trading Platform

ScrapX is a production-ready enterprise scrap trading platform built with React, TypeScript, Tailwind CSS, and Zustand.

## Highlights

- 100% standalone frontend architecture — zero backend or database setup required.
- Full local data state management with `localStorage` persistence.
- Complete Buyer, Seller, and Admin portals.
- AI-assisted price insights and listing description generation.
- Responsive, modern SaaS interface inspired by Stripe, Notion, and SAP.

## Running Locally

```bash
npm install
npm run dev
```

Build for production:
```bash
npm run build
```
